PDF工具箱使用说明

1. 运行 "PDF工具箱.exe" 启动程序
2. 主要功能：
   - PDF拆分、合并
   - PDF转图片、图片转PDF
   - 提取图片、文本、表格
   - PDF保护和解密
   - 批量打印
   - PDF预览

3. 系统要求：
   - Windows 7/8/10/11
   - 需要安装Poppler工具（用于PDF转图片）
   
4. Poppler安装说明：
   - 访问：https://github.com/oschwartz10612/poppler-windows/releases/
   - 下载最新版本
   - 解压到 C:\poppler 目录
   - 确保 C:\poppler\Library\bin 目录存在
